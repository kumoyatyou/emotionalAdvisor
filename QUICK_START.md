# 👶 小白专属：从零到一全流程使用指南 (Quick Start)

欢迎来到 **AI 情感军师 Agent**！如果你是第一次接触这种本地部署的 AI 项目，不要慌，这篇教程就是为你准备的。

只需要按照下面的 5 个步骤，你就可以在自己的电脑上拥有一个完全属于你的“情感分析大师”。

---

## 第一步：准备环境（只需做一次）

要让代码跑起来，你的电脑上需要装好 Python。

1. **安装 Python**
   - 去 [Python 官网](https://www.python.org/downloads/) 下载 Python (建议下载 3.10 或以上版本)。
   - ⚠️ **超级重点**：安装时，**一定要勾选 `Add Python.exe to PATH`**（添加到环境变量），然后再点击 Install Now。
2. **下载本项目**
   - 点击本页面右上角的绿色按钮 `Code` -> `Download ZIP`。
   - 下载后解压到一个你找得到的文件夹里（比如 `D:\emotionalAdvisor`）。
3. **安装依赖包**
   - 打开解压好的文件夹，在文件夹的**地址栏**里输入 `cmd` 并按下回车，这会弹出一个黑色的命令行窗口。
   - 在黑窗口里输入以下命令并回车，耐心等待它跑完（这会帮你装好项目需要的所有插件）：
     ```bash
     pip install -r requirements.txt
     ```

---

## 第二步：配置大模型（给 Agent 注入灵魂）

我们的项目只是一个“躯壳”，它需要连接到大模型（比如 Kimi、DeepSeek）才能拥有大脑。

1. 去 [硅基流动官网](https://siliconflow.cn/) 注册一个账号。
2. 在左侧菜单找到 **API 密钥**，新建一个并复制那一长串以 `sk-` 开头的代码。
3. 回到我们的项目文件夹，找到 `.env.example` 文件，把它重命名为 `.env`。
4. 用记事本（或任意文本编辑器）打开 `.env` 文件。
5. 按照下面的模板，把你的密钥填进去，其他的照抄就行
> 💡 *如果你想了解为什么这么配，或者遇到了配置问题，可以查看 📘 [环境变量 (.env) 配置完全指南](./ENV_CONFIG_GUIDE.md)。*

---

## 第三步：准备微信聊天记录（给 Agent 喂数据）

Agent 需要看到你们的聊天记录，才能帮你分析。目前最推荐的方式是使用开源工具 **WeFlow** 导出数据。

1. **下载 WeFlow**
   - 去 [WeFlow 项目主页](https://github.com/hicccc77/WeFlow/releases) 下载适合你电脑的版本并安装。
   - 启动 WeFlow 并按照它的提示登录微信。
2. **导出历史记录（推荐首次必做）**
   - 在 WeFlow 里找到你想分析的那个人的聊天记录。
   - 点击导出，**格式选择 JSON**。
   - 把导出的 JSON 文件，复制到我们项目的 `data/raw/` 文件夹里面。

*(注：如果你还想让 Agent **实时**看到对方发来的新消息，可以在 WeFlow 设置里开启“API 服务”和“主动推送”，然后把 WeFlow 提供的 Token 填到 `.env` 文件的 `WECHAT_SYNC_TOKEN` 里。小白第一次可以先跳过实时同步，只玩历史导出。)*

---

## 第四步：启动你的专属军师！

万事俱备，现在让它活过来。

1. 还是在刚才那个项目文件夹里，地址栏输入 `cmd` 回车。
2. 输入以下命令并回车：
   ```bash
   python launcher.py
   ```
3. 这个时候会弹出一个漂亮的终端界面，告诉你服务启动了。紧接着，你的浏览器会自动打开一个网页（默认是 `http://127.0.0.1:8000`）。

🎉 **恭喜你！这就是你的本地 AI 军师界面！**

---

## 第五步：开始第一次对话

在网页的聊天框里，你可以这样对它下达指令：

### 1. 建立档案并分析（必做）
> “我刚才在 data/raw 里放了一份聊天记录，帮我给对方建个档案，顺便分析一下她最近对我的态度怎么样？”

这时候，Agent 会自动去读取你刚才放进去的 JSON 文件，理解你们的聊天，然后生成一份详细的情感分析报告。

### 2. 寻求回复建议
> “她刚才说‘今天好累啊不想动了’，我该怎么回她显得比较体贴又不那么舔？”

Agent 会结合刚才建立的档案和性格分析，给你提供几套不同风格的回复话术。

### 3. 算一卦（体验八字 Skill）
> “我的生日是 1998年5月10日 早上8点，男，帮我算一下八字，顺便看看我最近的感情运势。”

它会自动调用 DeepSeek-V3 结合内置命理知识，给你排盘并给出通俗易懂的解读。

### 4. 模拟名人视角（体验角色模拟 Skill）
> “用马斯克的视角，给我这优柔寡断的感情状态一顿痛骂并给出建议。”

Agent 会瞬间切换人格，用极其犀利的口吻帮你“醍醐灌顶”。
